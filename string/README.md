# String (문자열)
1. String::new() 로 생성 가능하며 내부를 보면 Vec<u8> 이다.
2. 러스트의 문자열은 UTF-8로 인코딩되어 있다.
3. push_str(문자열), push(단일 문자) 추가 메서드도 기본적으로 제공
4. 러스트에서 문자열 +는 add 함수로 치환되어 사용된다
5. format! 매크로를 통해 복잡한 문자열을 조합하여 쓸 수 있다.
6. 영어는 1byte, 유럽/중동 2byte, 한글/한자/일어 3byte, 이모지/고대문자 4byte
7. 러스트에서는 인덱스를 이용하여 문자열 접근이 불가능하다.
8. 인덱스 범위를 지정해 문자열 슬라이스는 가능하다.
9. char 함수로 문자열을 반복도 가능하다.
10. bytes 함수로 원시 바이트 또한 가능
